# Pokemon_Move_Predictor

## Donde encontrar los reportes
Los reportes se encuentran en la carpeta "docs"

## Que hace cada archivo

export_json.js - Pasa un objeto de JS de la pokedex a a un JSON para poder usarlo en python

pokedex_data.js - Pokedex que usa Pokemon Showdown

pokedex.json - JSON de la pokedex de Pokemon Showdown

idx2pokemon.json - JSON para pasar de ID a nombre de pokemon

pokemon2idx.json - JSON para pasar de nombre de pokemon a ID

move_pred_model.keras - modelo

Move_Predictor_Arquitecture.ipynb - Codigo donde se encuntra las arquitecturas de los modelos

Move_Predictor_dataset.ipynb - Codigo para obtener los datasets.

Move_Predictor_Pipeline.ipynb - Codigo para probar el modelo

pokemons_vgc.csv - dataset de partidas pokemon

pokemons_vgc_normalized.csv - dataset de partidas pokemon con nombres en minusculas

pokemons_vgc_combined_normalized.csv - dataset de más partidas pokemon

pokemons_vgc_additional_normalized.csv - dataset de más partidas pokemon con nombres en minusculas
